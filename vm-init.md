# install nvm
get latest release https://github.com/nvm-sh/nvm/releases

``` bash
$ apt-get install bash-completion
$ curl https://raw.githubusercontent.com/creationix/nvm/v0.25.0/install.sh | bash
$ source ~/.profile
$ nvm install ${node-version}
```
